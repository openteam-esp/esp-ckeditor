= EspCkeditor

CKEditor 3.6.2 for rails >= 3.1.0 project with el_vfs_client (optional) support.

== Install

To Gemfile

  gem 'esp-ckeditor'

and

  $ bundle install

or

  $ gem install esp-ckeditor

== Usage

In layout or view include JavaScript

  <%= javascript_include_tag 'esp-ckeditor/application' %>

and use textarea with <tt>:class => :ckeditor</tt>
